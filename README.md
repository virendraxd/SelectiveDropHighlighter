# SelectiveDropHighlighter 🌟

A lightweight, client-side Fabric mod for Minecraft 1.21.11 that helps you keep track of the loot that matters.


## ✨ Features
* **Rarity-Based Outlines:** Dropped items glow with colors corresponding to their rarity (Common, Uncommon, Rare, Epic).
* **Smart Selection:** Toggle highlights for specific items directly from your inventory.
* **YACL Integration:** A clean, searchable settings menu to manage your highlighted items list.
* **Performance Friendly:** Built with optimized mixins to ensure zero impact on your FPS.

## ⌨️ Keybinds
| Action | Default Key | Description                                                                                    |
| :--- | :---: |:-----------------------------------------------------------------------------------------------|
| **Toggle Highlight** | `H` | Hover over an item in your inventory and press this to add/remove it or press H key to toggle. |
| **Open Menu** | `O` | Opens the Selective Drop Highlighter settings screen.                                          |

*Note: Both keys can be rebound in the standard Minecraft Controls menu.*

## 🛠️ Requirements
* **Fabric Loader** (0.18.4+)
* **Fabric API**
* **YetAnotherConfigLib (YACL)** (v3.8.2+)

## 🚀 Installation
1. Download the latest `.jar` from the [Modrinth](https://modrinth.com/mod/SelectiveDropHighlighter) page.
2. Drop it into your `.minecraft/mods` folder.
3. Ensure you have the required dependencies installed.
4. Launch the game and start highlighting!
